Q / K / V · 04

Token 的三重化身

点击向量 $\mathbf{x}$，观察它如何分裂为三种计算角色——
这是注意力机制运转的核心设计。

输入向量
$\mathbf{x}$
点击激活分裂
$\mathbf{q}=W^Q\mathbf{x}$ · $\mathbf{k}=W^K\mathbf{x}$ · $\mathbf{v}=W^V\mathbf{x}$
三组独立权重矩阵，赋予同一向量三种语义角色
🔦
Q — Query 查询向量
手持手电筒，主动搜寻——"我在找谁？"
q = WQx
🏷️
K — Key 键向量
举着身份牌，等待匹配——"我是谁？"
k = WKx
📦
V — Value 值向量
携带真实信息包——"我包含什么？"
v = WVx
